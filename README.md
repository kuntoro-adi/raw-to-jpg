# raw-to-jpg
Raw image to jpg.

## Format
Supported raw formats: DNG and NEF.

## Usage
1.  Modify these lines 
```
# Directory of image source.
SOURCE_DIR = '../Datasets/raw-images'
# Directory of target location.
TARGET_DIR = '../Datasets/raw-jpgs'
```
2. Run `python convert_raw_to_jpg.py`
